# Data Analysis Portfolio

This repository contains a collection of data analysis projects developed for learning, practice, and portfolio purposes.  
The projects focus on exploratory data analysis, data cleaning, visualization, and insight generation using real-world datasets.

The main goal of this portfolio is to showcase my analytical skills, workflow, and ability to communicate insights through data.
